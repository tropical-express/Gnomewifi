Gnomewifi

What is a managed Wi-Fi network?

A managed Wi-Fi network is a network that your administrator or school pre-installed into your Chromebook. To know if a Wi-Fi network is managed, check if that network has a symbol like this next to it:

Managed networks connect you to a DNS server with restrictions. Some features may not work even on sites that are normally unblocked.

What is Gnomewifi?

Gnomewifi is an exploit that allows Chromebook users to remove all restrictions from a managed Wi-Fi network. It works similarly to another exploit, CAUB.

How does it work?

Managed networks prevent you from changing DNS servers. Gnomewifi works by creating a new network configuration that replaces the managed network with one using unrestricted DNS servers. This allows access without restrictions.

How to use Gnomewifi:

  • Go to chrome://network#state and scroll to "Favorite Networks".
  • Find the managed network you want to modify and click the + sign next to it.
  • Copy the text that appears. You can copy multiple networks at once.
  • Paste the copied text into the textbox below.
  • Press the "Download" button to download the Gnomewifi file.

After downloading, go to chrome://network#general and click "Import ONC" to import the file.

Congratulations, you have set up Gnomewifi!
